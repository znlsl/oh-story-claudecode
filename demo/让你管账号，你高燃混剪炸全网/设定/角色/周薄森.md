# 周薄森

## 基本信息
- 身份：火箭军文工团副团长，文职6级（相当于少校），分管新媒体宣传工作，江晨直属上司
- 核心特质：慈父型护短领导 + 爱面子会甩锅 + 嗅到好处立刻变脸
- 当前能力：体制内政治嗅觉敏锐，善于上下周旋；对新媒体本身完全外行（第10章看专业版成片后率先开口"缺了灵魂"，是感性判断而非专业分析）
- 核心动机：维护本单位宣传成绩、保住年中评比排名，顺带护住自己手下的兵不受连累
- 弱点/缺陷：遇事先想"出事了怎么甩锅"；吹捧江晨时被当场拆穿从没看过视频，老脸挂不住；明知有舆论风险（第13章）却犹豫不决，错失主动干预窗口

## 外在表现

**身份/职位**：火箭军文工团副团长，军装笔挺，举止老持，习惯端茶杯、揉太阳穴（第6章：揉着太阳穴步履沉重走回办公室，端起茶杯发现茶水凉透了）。

**言行标志**：
- 情绪管理一般——紧张时硬撑镇定，得知好消息后当场"脸上的笑容都快咧到耳根了"，拍大腿、"连连拍着大腿"
- 护短行动：第一反应不是查江晨的错，而是让江晨躲起来，自己顶上去揽责（"要是真涉及泄密，责任不在江晨"）
- 甩锅习惯：风声不好时主动接锅是为了掌控主动权，不是真无私——一旦确认是好事，立刻转口"这孩子是我们团的骄傲"
- 第10章：专业版成片播放后全场沉默，他率先开口点出"缺了灵魂"，替张耀祖说出众人的想法，有一定军事审美直觉

**外貌**：（原文未详细描写，占位）中年男性，军装，面相老持。

## 内在分析

**性格**：
- 体制老鸟，官场反应比内容判断快——第6章得知是来表彰不是来查的，0.5秒变脸，从"出事了怎么办"直接跳到"江晨是我们团台柱子"
- 对下属有真实的庇护本能（第6章独白"毕竟是自己手下的兵，无论如何都要把江晨护住"），但这种庇护掺杂了面子和利益考量，不是纯粹的父爱
- 第15章被逼到墙角时敢于联手张耀祖越级向军长汇报为江晨争取时间，说明在关键时刻能压过习惯性自保

**目标**：短期——年中评比不垫底；中期——借江晨的宣传成绩洗掉"文工团新媒体为零"的烂底子

**秘密/内心矛盾**：吹捧江晨时被钟嘉嘉当场见证两次变脸，自己心知肚明从没认真看过江晨的视频，专业判断全靠拍感觉——这是他底气不足、一直跟着张耀祖走的根本原因

## 语言风格

**说话方式**：
- 对下属：短促命令式，不废话（"没规矩！""快，去通知江晨，让他躲起来，别露面！"）
- 对上级（张耀祖）：言简意赅汇报，顾不上礼数时直接开门见山
- 对外来人员/记者：官方客套 + 飞速变脸，一旦确认没麻烦就立刻切换成热情推销模式，夸人滔滔不绝
- 专业判断时：凭感觉出口，一句话定性（"缺了灵魂"），不解释原因，但往往说中要害

**口头禅/标志表达**：
- 遇急事："出事了！""这事儿要是捅出去，后果不堪设想。"
- 夸江晨时的套话模板："这孩子年轻有为，有想法，有脑子""镜头语言独特""前途无量""是我们团的骄傲"
- 第10章风格金句："缺了灵魂。"（短句，一锤定音）

**与不同人说话的区别**：
| 对话对象 | 语气特征 |
|---------|---------|
| 江晨（下属） | 命令式，但带隐性庇护；私下比当面宽松 |
| 张耀祖（上级） | 服从为主，关键信息及时上报，偶尔先斩后奏（如先叫江晨躲起来再打电话） |
| 钟嘉嘉（外部记者） | 从警惕到热情，变脸清晰可见；夸人时停不下来，有点社牛 |
| 齐首长（更高上级） | 原文未直接呈现，推测毕恭毕敬 |
