# 钟嘉嘉

## 基本信息
- 身份：解放军报实习记者；爷爷为退休高级军官（齐首长系军方背景，爷爷层级与之相近，原文未明确对应关系，待复核）
- 核心特质：职业本能敏锐、对江晨情愫早生、敢借家族资源主动助力
- 当前能力：军报采访资质（可独立带车入营）、军方人脉（通过爷爷渠道推送宣传）、视频出镜（第16章扮演新时代女兵，独白旁白压轴）
- 核心动机：做出有分量的军事报道；对江晨的情感从职业好奇渐向个人倾慕演变
- 弱点/缺陷：以"工作需要"掩盖私心（第8章对爷爷说"保准没私心"时脸红），情感上主动但不敢挑明

## 外在表现

**身份/言行**
- 实习记者身份，但配车、随行兵配置明显超出实习生规格，张耀祖察觉"来头不简单"
- 主动性强：被告知用微信发资料，直接挂电话亲自跑来（第12章）
- 采访风格利落，掏出笔记本就开始深挖，不绕弯
- 借爷爷资源时嘴上喊"工作需要"，实际立刻脸红收不住

**外貌**（原文有限细节）
- 修长手指（第8章"修长的手指轻点屏幕"）
- 笑容甜，习惯发可爱表情包（第8章首条微信发小狗表情）
- 第16章出镜：英姿飒爽的新时代女兵造型，军装整洁干净，背影挺拔

## 内在分析

**性格**
- 外热内细：表面爽快活泼，实则察言观色（第8章偷看爷爷反应、看准时机才开口请求）
- 有分寸感：喜欢江晨却不当面表态，用"工作"当遮掩层
- 情感真诚：第17章数据暴涨时热泪盈眶，第20章专程跑来送老兵礼物，不是做样子

**目标**
- 短期：把江晨的故事做成有分量的军报专题
- 长期（隐线）：与江晨关系从搭档走向更近——原文每次相见都有情绪升温节点

**秘密**
- 爷爷的军方背景和实际级别原文未明写，仅知退休高级军官、有军政影响力、能推动官方宣传渠道；续写前需复核是否与齐首长存在关联

## 语言风格

**说话方式**
- 对江晨：轻松带点俏皮，第一条微信发小狗表情＋"江晨同志"正式称呼叠加，温度藏在用词错位里
- 对爷爷：撒娇型，"诶呀爷爷你别管""保准没私心"，习惯用语气词软化请求
- 正式场合（采访）：干脆直接，掏本子就问，不废话
- 情绪顶点（第16章旁白）：字斟句酌，铿锵收尾——"这盛世，如您所愿！"

**口头禅/标志动作**
- 偷偷看对方反应再开口（爷爷面前、江晨面前均有）
- 脸红时习惯低头或转移话题

**与不同人说话的区别**
| 对象 | 风格 |
|------|------|
| 江晨 | 俏皮暖场→认真讨论，情绪递进，信息量大 |
| 爷爷 | 撒娇＋借力，掌握节奏，目的性强 |
| 上级领导（张耀祖等）| 礼貌简洁，保持记者距离感 |
