## 龙血战士家族的秘密

| 项目 | 内容 |
|------|------|
| 标题 | 龙血战士家族的秘密 |
| 类型 | 主线 |
| 概要 | 乌山镇护卫队长、六级战士希尔曼晨练时发现六岁的林雷·巴鲁克坚持远超同龄人，夸他像个男人，又以四大终极战士与圣域强者传说点燃林雷向往，林雷主动加入更大龄团队训练。半年后参军时节，父亲霍格换上华贵衣服，带林雷与管家希里进只用于祭祀的宗堂，在密密麻麻的祖先灵位前揭示巴鲁克家族传承五千年、正是蕴含龙血战士血脉的古老家族。霍格用龙血针为已满六岁半的林雷检测血脉浓度，结果未达成为龙血战士的条件，林雷落泪自责；霍格安慰他并道出真正心愿——夺回六百年前因家族衰败被以十八万金币变卖的传承之宝战刀"杀戮"，林雷因此握紧拳头立誓将来带回。此后两年间林雷靠肉体苦练（静态张力锻炼），并在父亲严苛的政治地理艺术教育下成长；目睹父亲变卖石雕"猛狮"换钱后想学石雕改善家境被否决，命去清洁宗堂；林雷提前完成清洁后到后园古屋寻宝，愤掷木杆撞塌腐朽木架，一颗黑色戒指从夹层跌落，他踩到后捡起，认作"龙"命名为盘龙戒指，穿成胸坠戴在胸前，赶赴傍晚训练。 |
| 核心目标 | 认知家族龙血战士血脉传承、确立林雷"成为强者重现家族荣耀"的人生目标 |
| 核心冲突 | 家族五千年龙血战士辉煌 vs 当代血脉稀薄检测失败、经济困顿（变卖石雕度日、战刀杀戮被变卖六百年未赎回） |
| 桥段标签 | 扮猪吃虎潜质，护短，机缘逆天 |
| 章节范围 | 第1-7章 |

结构分布：
- 铺垫期：第1章（晨练立人设、四大终极战士传说、林雷立志）
- 发展期：第2-5章（宗堂揭血脉、龙血针检测失败、立誓夺回战刀、肉体苦练、严苛教育、变卖石雕）
- 高潮期：第6章（古屋寻宝、木架坍塌、黑戒跌落）
- 收尾期：第7章（拾戒命名盘龙戒指、穿成胸坠、东方传来地面震动埋钩）
