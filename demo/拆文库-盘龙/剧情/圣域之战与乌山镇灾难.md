## 圣域之战与乌山镇灾难

| 项目 | 内容 |
|------|------|
| 标题 | 圣域之战与乌山镇灾难 |
| 类型 | 危机 |
| 概要 | 迅猛龙过境次日清晨，孩子们还在谈论昨日战斗、林雷如实讲解迅猛龙属性，晨练刚结束，东方两三百米高空出现一条长达百米的黑龙。黑龙头顶站着圣域魔法师鲁迪（灰袍人），与凌空而立、背负长剑的圣域剑圣蒂隆（绿衣中年人）对峙。希尔曼判断圣域强者即将动手、能量波及足以让无遮蔽的孩子丧命，厉声驱赶所有孩子回家躲藏，霍格派战士协助疏散，命林雷躲进库房地下室。蒂隆与鲁迪开战：蒂隆以碧绿斗气护体、挥数十米剑芒，鲁迪靠钻石般护体战甲免伤，随即召出密密麻麻房屋大小的巨石砸向蒂隆，蒂隆斗气爆发将巨石全部震裂，近五分之一碎石从高空朝乌山镇四散迸射，灾难降临。林雷奔向库房途中目睹两米长巨石射向门口的四岁弟弟沃顿，不顾落石飞扑压住沃顿，库房屋顶墙壁被砸塌，管家希里斗气爆发飞扑护住，三人被掩埋。蒂隆破净巨石后落石停止，碎石堆被破开，林雷以身护住沃顿、自己头部流血，沃顿毫发无伤，霍格落泪、希里包扎。林雷头部血染胸前盘龙之戒，黝黑材质缓缓吸血、表面隐隐闪烁朦胧光影，被衣服遮挡无人察觉。鲁迪召出地系十级禁忌魔法大地守卫（土黄色巨人）与蒂隆缠斗，蒂隆重伤后引爆所争之物使双方都得不到、化光逃向东北，鲁迪认出蒂隆是星空剑圣暂不能杀、乘黑龙离去。全镇死亡三百余、伤近千，约占五千人五分之一，霍格因家族经济困顿无力救助、命希尔曼率护卫队巡查。 |
| 核心目标 | 林雷在圣域之战的灭顶灾难中保全弟弟与自身性命，并触发盘龙之戒吸血激活 |
| 核心冲突 | 两大圣域强者（鲁迪/蒂隆）毁天灭地的交战 vs 毫无抵抗力的乌山镇平民与孩子的生存 |
| 桥段标签 | 护短，绝境翻盘，机缘逆天 |
| 章节范围 | 第14-19章（觉醒承接第18-19章） |

结构分布：
- 铺垫期：第14章（孩子谈昨日战斗、林雷讲迅猛龙、晨练结束黑龙现身）
- 发展期：第15章（鲁迪蒂隆对峙、龙威压场、希尔曼驱散孩子、林雷躲库房）
- 高潮期：第16-17章（巨石震裂落石成灾、林雷飞扑护沃顿被埋、头部流血染戒吸血、希里包扎、大地守卫现身）
- 收尾期：第18-19章（蒂隆引爆逃走、鲁迪离去、全镇伤亡统计；盘龙之戒夜飞出化德林柯沃特、滴血认主、应允授魔法）
