## 强者过境与魔法启蒙

| 项目 | 内容 |
|------|------|
| 标题 | 强者过境与魔法启蒙 |
| 类型 | 主线 |
| 概要 | 引起地面震动的庞然大物现身，是七级魔兽迅猛龙，背上盘坐一位紫袍魔法师。希尔曼护住孩子、应答确认此处是乌山镇后尾随跟去，林雷不顾劝阻留在希尔曼身边旁观。紫袍魔法师欲用七百金币强购价值十万金币的德佩洛影钻，被芬莱王国凯莱家族的七人佣兵小队拒绝，遂下令迅猛龙喷火开战。战士用斗气护罩与冰之守护战甲抵御、踢巨石攻击，但战士路加被龙尾卷入龙口咬碎、半截大腿跌落；林雷目睹血腥后先呼吸困难、继而血液沸腾产生杀戮渴望，疑为龙血血脉所致。紫袍魔法师释放八级火系魔法火蛇之舞，七条数十米火蛇将佣兵小队连人带魔兽尽数烧成灰烬、焚毁乌山镇大量石屋，又劈闪电击落逃走的女弓箭手；他从灰烬中拾得寻找十年的德佩洛影钻，称要嵌入法杖对付海曼斯，并将融化的金币作为对乌山镇的赔偿后骑龙离去。事后林雷独坐卧室渴望成为魔法师、拥有魔兽而辗转难眠，霍格夜访赞其具龙血战士品质，并逐一讲解魔法师与战士的实力地位差别、天地元素修炼、收服魔兽需击败它并布置灵魂契约魔法阵（或一万金币卷轴）等知识。林雷暗下决心走成为七级魔法师再收魔兽之路，霍格准许他深秋赴王都芬莱城参加魔法学生测试。 |
| 核心目标 | 林雷确立"成为魔法师、收服魔兽"的新方向并获准赴芬莱城测试天赋 |
| 核心冲突 | 林雷对强者力量（火蛇之舞/迅猛龙）的向往 vs 战士道路的成长天花板、龙血血脉抵触斗气、家族经济无力支撑收服魔兽 |
| 桥段标签 | 围观震惊，机缘逆天 |
| 章节范围 | 第8-13章 |

结构分布：
- 铺垫期：第8章（迅猛龙现身、紫袍魔法师确认乌山镇、罗瑞讲解迅猛龙实力）
- 发展期：第9-10章（佣兵拒交影钻开战、战士斗气护罩反击、路加被咬碎、林雷血液沸腾）
- 高潮期：第11章（火蛇之舞屠灭佣兵小队焚城、魔法师拾影钻、金币赔偿离去）
- 收尾期：第12-13章（林雷渴望成魔法师、霍格讲魔法师/收魔兽知识、获准深秋测试）
