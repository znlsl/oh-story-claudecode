## 故事梗概

沈暮月是沈家私生女，三年前被人做局和霍氏继承人霍庭煜送进同一个房间，生下儿子阿忆。京圈都说她设计灌醉霍庭煜、费尽心思想当霍夫人，霍庭煜也这么认定，把她和孩子当成人生污点，当玩物随叫随到。一次门口偷听到"私生子而已"，又被半夜闯进卧室的霍庭煜召之即来，沈暮月头一回正面拒绝了他，决意带孩子逃。

霍庭煜赴海城那月，沈暮月借一场车祸火灾死遁（原文以"早间新闻"的意外形式呈现，霍庭煜后来点破是"拿假死做局"），带阿忆和奶妈杨姨南下春城，开鲜花饼店重新生活。半年后霍庭煜回京确认"死讯"，从白布里露出她生日缠着他买的玉镯，他当场失态。此后他禁止任何人提起母子，却也没真答应家里安排的相亲。

霍氏要开辟西南市场，霍庭煜来了春城，深夜抱着阿忆出现，揭穿假死。他强行纠缠、留宿、暗中监控盯着她和邻居徐宇一家的来往。期间他神秘失踪，堂弟霍宁为夺继承权持枪闯进沈暮月家。危急中沈暮月自污撇清、谎称阿忆不是霍家血脉，霍庭煜带人赶到，掐住她的脖子要她把话再说一遍。

她把话说死了——这段时间都在敷衍他、只为逃离，从前得不到的爱以后也不要了。霍庭煜这才正视自己：三年前的事他早原谅了，他深爱着她，想重新开始。可他给出的还是"我可以给你霍太太的身份"这种施舍口吻，被沈暮月两次拒绝。阿忆把他推出门，门关上。他在门外守了很多天，最后托阿忆带回一封绝交信，告诉孩子"遇到危险来找这个叔叔"，转身离开。五年后，霍庭煜终身未娶，传闻有一个儿子，是后来霍氏集团的接班人。

### 前置反转检查

### 主反转：认知反转

**反转类型**：认知反转——读者（和女主）对霍庭煜"冷漠、无爱、只有占有欲"的整体认知，被颠覆为"一直深爱、早已原谅、夜不能寐"。它翻转的是读者对这个男人的整体感情色彩：从"冷漠无爱"翻成"一直深爱、早已原谅"，这一层比单条信息的更正更重。

**反转机制**：

- **铺垫线索（6 条，认知反转的证据链）**：
  1. N3 旁人提议"给我们玩玩"，霍庭煜冷声"不配"——护短，不舍得。
  2. N16 接到"死讯"淡然吩咐处理后事，"却胸口一阵绞痛"——身体先于嘴诚实。
  3. N18 玉镯（她生日缠着他买的）从白布里露出，"双目赤红""怎么可能"——爱被实物坐实。
  4. N32 女主说"你不是要结婚了"，他反问"你听谁说我结婚的"——他没真要娶别人。
  5. N38 监控前看她和徐宇一家，冷笑"还以为是一家三口"——吃醋。
  6. N41–N45 堂弟霍宁拿她当饵逼霍庭煜现身，他带人赶到、亲手护住她——危急关头先护人，藏不住的在乎。
- **误导方向**：第一人称把女主的认知"他不爱我、他要结婚、他只有占有欲"原样交给读者，让读者跟着误判。
- **真相揭示**：N47 切到第三人称内心独白，集中倒出"三年前的事他原谅了""他深爱着沈暮月""夜里根本无法入眠"。
- **合理性**：回看成立，6 处铺垫全对得上。弱点在于释放方式——靠一大段独白直给，缺过程展示。

**时机**：约 93% 位置 | 铺垫:释放 ≈ 9:1（铺垫贯穿全文，释放挤在结尾）

**效果**：惊喜度 3 / 5（铺垫太足，读者早猜到"他其实爱你"，反转更多是"果然如此"的确认）| 合理性 4 / 5 | 情绪冲击 4.5 / 5（冲击不来自"竟然爱"，来自"爱也没用了"）

> 这篇的反转价值落在"意料之中的意难平"，不在"出人意料"。读者从 N16 起就拿到答案，看的就是女主因为不知道而越走越远——揪心大于惊讶，这是追妻火葬场的标准打法。

## 四、写作手法（摘要）

完整 11 项见 `写作手法.md`。核心 5 项：

| 手法 | 位置 | 效果 | 可复用度 |
|------|------|------|----------|
| 第一人称信息差锁"爱不爱" | 全文 | POV 锁死女主，男主的爱只用物件 + 身体反应 + 第三人称插入漏给读者，制造"皇帝不急太监急" | 极高 |
| 物件钩子（玉镯）| N18 | 生日缠着买的玉镯一次性引爆，比十句心理描写更能坐实隐藏的爱 | 高 |
| 童言补刀 | N12 / N50 / N53 | 不懂事的孩子说最扎心的话："没有爸爸""是爸爸又欺负你吗""爸爸再见" | 高 |
| 二号男主对照 | N24–N39 | 徐宇（温和、给阿忆父爱）反衬男主迟到，并制造男主危机感（监控吃醋）| 高 |
| 自污式护子 | N43–N44 | 危急中女主主动把自己说成玩物、把孩子说成"野种"，以退为进，比硬刚可信 | 高 |

**POV 策略**：第一人称（女主）为主，§03/§04/§09 三处切第三人称写霍庭煜，专门用来漏男主藏起来的爱；代价是男主转变缺过程、靠独白直给。
**对话手法**：对话占比约 55%（标准），潜台词率约 65%（高）。模式：压制式（男主短句碾压）+ 信息差式（女主当霍宁面说假话，三方理解各异）+ 推拉式（女主对徐宇）。
**手法总计**：11 项 | 核心 Top3：①信息差锁"爱不爱" ②玉镯物件钩子 ③童言补刀 | 创新点：把"死遁/假死"嫁接进追妻火葬场。

### 可复用结构（5 条）

1. **死遁嫁接追妻火葬场**：用"假死"给女主的"不回头"加一道物理隔断（你以为我死了），男主的追悔代价被放大。适用场景：追妻 / 虐渣 / 世情。失效点：假死逻辑要兜圆（尸体辨认、户籍），否则全篇崩。
2. **第一人称信息差锁"爱不爱"**：POV 锁死女主，男主的爱只用物件 + 身体反应 + 第三人称插入漏给读者，制造"皇帝不急太监急"。适用场景：所有追妻 / 双向错位暗恋。失效点：男主信号给太满会提前泄底，给太少读者不买账，控制在 2–3 处。
3. **物件钩子坐实隐藏情感**：一个有来历的小物件（玉镯=她生日缠着买的）一次性引爆，胜过十句心理描写。适用场景：需要证明"他其实在乎"的所有题材。
4. **童言补刀**：用不懂事的孩子说最扎心的话（"是爸爸又欺负你吗"）替成人挡情绪、加痛。适用场景：带娃虐文 / 亲子向。
5. **留白结尾 + 时间跳跃**："五年后……终身未娶……有一个接班的儿子"，把团圆 / 意难平交给读者脑补。适用场景：BE / 开放式收尾。

### 同类型写作动作

写下一篇追妻火葬场短篇时：① 前 40% 用一个具体羞辱事件（对标"私生子而已"当众说）把女主憋到读者想替她走；② 男主的爱全程禁止女主视角直说，只用 2–3 个物件 / 反常细节漏给读者；③ 女主至少明确拒绝两次（对标 N48 施舍式身份被拒 + N51 告白被拒），第二次配一段"清算式"长台词；④ 结尾用时间跳跃 + 留白定调，不写团圆也不写复合。
