# 短篇写作通用底座

> 短篇网文正文写作的公共父类。服务番茄 / 七猫 / 黑岩 / 盐选通俗短篇（1-2 万字），不是传统文学。
> 题材风格包（`genre-styles/`）在此基础上做覆盖加味；本文件管所有题材共通的原生写法。
> **总原则：情绪宁烈不温。短篇是情绪产品，靠直白共鸣留人，不靠克制留白。**

---

## 1. 人称与腔调：第一人称受害者，双轨切换

- **默认第一人称「我」全程主导**。受害者亲历视角，读者直接钻进主角身体和脑子里。他人视角（渣男 / 白月光 / 反派）只作**追悔 / 得意的补刀短插**，不平分篇幅。极少数男频自救文用男「我」。
- **腔调是双轨，靠「心死」开关切换**：

| 阶段 | 声线 | 怎么写 |
|------|------|--------|
| 受虐段 | 直白宣泄 | 喊疼、喊委屈，眼泪不争气，心如死灰。成段内心独白和连环反问 |
| 反击 / 反转段 | 冷静审判 | 抽离、反讽、居高临下。短句见血，「呵」「我纯粹觉得你比较可笑」 |

- 「心死定格句」是切换枢纽：撞破真相后用一句独立成段的心死宣告（「那一刻，我的心彻底死了」），把情节从委屈隐忍切到冷静决裂，后续行为逻辑全由它翻转。

---

## 2. 情绪外化：直接写情绪词，后面接一个场景里的具体动作（替代机械替换表）

短篇最核心的写法，也是和长篇审美冲突最大的一条。

真实番茄里，情绪词（心如死灰、心如刀绞、几欲作呕、眼泪不争气）就是直接写出来的，而且用得很密——这是卖点，不是 AI 味。关键在后面一句：**情绪词后面要接一个这个场景里此刻特有的具体动作或物件，不能让情绪词单独飘着。**

照真实小说写（以下都是原文，照这个语感）：

- 《为给白月光出气，男友群发我私密照》我心如死灰，泪水早已模糊了视线。
- 《为给白月光出气，男友群发我私密照》我紧紧抿着嘴唇，直到尝到血腥味，才勉强找回一丝清明。
- 《被凌辱后，我把他们送上法庭》我在一旁看得心如刀绞，几欲作呕。
- 《闻笙有两意》摸到那碗凉掉的排骨汤时，我的心更冰冷了。上面还浮着凝固的油，根本喝不了。
- 《和闺蜜死遁后，文武状元悔疯了》怒火和心痛在胸腔翻涌。我鼻头一酸，将玉佩摔得支离破碎。
- 《为讨真千金欢心》我紧紧攥着衣角，颤抖着问。

看这些原文的共同点：情绪词后面接的是「此刻手里、眼前那个具体的东西」——凉掉的排骨汤、摔碎的玉佩、攥着的衣角、抿出血腥味的嘴唇。不是哪篇都能套的通用生理反应（单写「指甲掐进掌心」「心猛地一沉」「鼻头一酸」就空）。

**要删的两种：**

1. 后面接不上任何具体东西、单独出现的情绪总结句（「一丝悲伤涌上心头」「感到深深的绝望」「仿佛整个世界失去了颜色」）。
2. 同一个情绪反复写、写完再补一句解释它多疼。一个情绪写一次、接一个具体动作，就往下走。

## 3. 叙述姿态：在场叙述者（区别于长篇深度限知）

短篇叙述者**高度在场**：可以跳出当下替读者定性、剧透、下判词、预告火葬场。这是爽感引擎，不是 AI 味。

- 留：第一人称主角的**主观审判句**（「原来我以为的爱只是一场笑话」）、**复仇者剧透勾子**（「他们不知道，我已经是西域摄政王」）、**火葬场前瞻预告**（「我还不知道，他正发疯似的找我」「可后来他后悔了」）。
- 删：真·中立全知摄像机腔——和角色情绪无关、纯交代背景的作者解说。

判据：这句跳出当下，是**带主角主观偏色的审判 / 勾追读**，还是**中立无情绪的作者讲解**？前者留，后者删。

## 4. 小节结构：超短章节制，一节一炸点

按平台两套节奏：

| 平台 | 小节标记 | 段落 | 单节字数 |
|------|----------|------|----------|
| 黑岩 | `###` + 数字（`###1.`） | 一句一段，多换行留白 | 800-1200 |
| 番茄 / 七猫 | `###第N章` 或 `###1.` | 短段为底色，允许多句回溯叙述段交织对话 | 1000-2000（古言 2000-4000） |

- 每个小节：**一个背叛升级 / 一次态度反转 + 一句更狠的台词 + 一条读者新获知信息**。
- 章尾**不收束情绪、必留钩**。
- **疏密**：背叛 / 打脸 / 反转 / 决裂处详写慢放，逐拍展开；过场 / 赶路 / 信息交代一句带过，把字数预算让给情绪节点。

---

## 5. 钩子：极高密度，阶梯式背叛

- 导语 50-300 字塞 3-5 个事件 / 反转；正文几乎每节一个新爆料或一次态度反转；黑岩密到每 3-5 句一拐。
- **阶梯式升级**：道德踩踏 → 经济 → 生命，越揭越大。一页里敢叠多重雷（绝症骗局 + 当众拥吻 + 孩子药引…），别单点反复抒情。
- **爽点章章前推**：每章必须推进一个反打 / 行动 / 亮牌（扇耳光、取消婚礼、删 B 超单转身走、当众揭穿、亮身份），不要停在情绪闭环；章尾留反杀预告（「已联系律师」「已查到他的致命把柄」），制造追读缺口。读者要的是「下一章想看她怎么打脸」。
- **当众围观放大羞辱**：把背叛 / 打脸放在有外人见证的场合（宴席、股东会、群聊截图、请安），用众人的起哄 / 鄙夷 / 沉默放大屈辱与出气感；独角戏起不来「丢脸感」。
- 共通钩子母题（题材包细化）：白月光 / 小青梅触发背叛、偷听 / 撞破真相一次性砸、灾难期求救反被辱、朋友圈 / 物证坐实、数字 / 年限承重再清零、身份反差底牌、当众羞辱 + 反转倒计时。

---

## 6. 对话：占比高，三类台词撑全篇

- 占比约 40-60%（盐言 / 黑岩打脸篇可到 60%；抒情虐 / 心死篇可低到 30-40%）。**按情绪驱动还是事件驱动浮动，不设死线。**
- 三类台词：① 施害者道德踩踏 / PUA 狠话（钩子本体，越理直气壮越好）；② 白月光「娇弱委屈」软刀（对不起 / 我不是故意的 / 都怪我）；③ 主角心死后冷静审判金句。
- 权力博弈靠对话长度反转标记：施害方前期长篇 PUA，主角后期一句话冷怼。详见 `dialogue-mastery.md`。

---

## 7. 收尾：火葬场预告 / 审判金句 / 漠视离场

短篇收口都不是文学升华：

| 收口式 | 例 |
|--------|-----|
| 火葬场反转 | 主角走后施害者悔疯 / 悬赏寻找，主角已无波澜或冷拒（「敦宜，你不要我了吗？」） |
| 审判 / 反讽金句盖棺 | 「迟来的深情比草还贱」「在他们眼里我是锦绣上的破洞」「晏家，咱们的账慢慢算」 |
| 转身离场 / 净身出户 | 签离婚协议、改嫁、带孩子一去不归、亮身份碾压 |
| 漠视收尾 | 「那是他们的故事了，与我再无瓜葛」——比报复更狠 |

- 删：空洞哲理升华（「人生就是这样」「她终于明白了」）。
- 留：第一人称复仇预告 / 审判金句 / 心死余韵——这是读者截图传播的高光。

## 8. 节奏与标点

- 句短、段碎，手机流式。短句 = 情绪密度高。
- 标点跟语气走：质问用问号，爆发峰值少量感叹；犹豫 / 未尽 / 被打断用短句、动作或换行断句。
- 三连排比是抒情堆苦难的手法（「在暴雨夜、在手术室门口、在我父亲的葬礼上」）：**有功能的递进 / 枚举保留**，只砍无信息量的纯修辞凑数。
- **标点硬约束**：本仓库正文输出统一不出现 `……` / `——` / `—` / `--`（去 AI 脚本与 hook 强制收口）。停顿、哽咽、未尽、留白一律用短句、句号、逗号、动作 beat 或换行实现。题材包范例中若保留 `……`，只为示意手法，落到正文要按本条改写。

## 9. 与长篇的根本区别

| 维度 | 长篇规则 | 短篇底座 |
|------|----------|----------|
| 视角 | 深度限知、不剧透 | 第一人称在场、主动剧透火葬场 |
| 情绪 | 身体动作替代情绪词 | 情绪词直接写 + 后接具体反应 |
| 章尾 | 禁总结 / 升华 / 预告 | 审判金句 / 火葬场预告是卖点 |
| 结构 | 贯穿道具三次 / 数字阶梯 | 偷听 / 底牌 / 回旋镖 / 次数累积 |
| 内心 | 独白 ≤ 2 段 | 成段宣泄独白 + 连环反问是主引擎 |
| 排比 | 砍到一条 | 抒情堆苦难有功能则留 |

## 10. 场景写法：三维度揉进

每个子事件不是一句话就完，包含三个维度，**织进同一段连续正文，不按维度分段**：

| 维度 | 内容 | 要点 |
|------|------|------|
| 发生 | 事情出现了 | 1-2 句，含一个具体细节 |
| 感知 | 主角注意到什么 | 至少 1 个感官细节，聚焦一个物件或身体部位 |
| 反应 | 身体如何回应 | 具体身体动作（手 / 眼 / 呼吸 / 姿势），可含一句极短心理定格 |

反模式（绝对禁止）：把三维度拆成三段依次写完，同一件事掰开写三遍。

- 正确（揉进式，《闻笙有两意》原文）：我以为陆池是真心跟我道歉，但摸到那碗凉掉的排骨汤时，我的心更冰冷了。上面还浮着凝固的油，根本喝不了。（摸汤、凝固的油、心冰冷揉在一段里）
- 错误（把上面这段拆成三段堆叠）：我摸到那碗排骨汤。／ 汤是凉的，浮着凝固的油。／ 我的心更冰冷了。（同一件事掰成三句，读者看三遍）

**画面分段**：揉进不等于一段到底。按新动作 / 新物件 / 新信息 / 新对话 / 视线转移断段，不按维度断段。一段承载一个镜头拍点或一条连续推理 / 氛围 / 情绪链。

## 11. 疏密分配 + 一动一静

**疏密**：不要每个 beat 一样长一样细（平均用力是 AI 腔的根）。

| beat 类型 | 笔墨 |
|---|---|
| 爽点 / 打脸 / 反转 / 情绪高潮 | 密（详写，感知动作对话铺满，慢镜头逐拍） |
| 过场 / 赶路 / 信息交代 / 时间跳转 | 疏（1-2 句带过，不展开三维度） |
| 铺垫 / 日常 / 关系升温 | 中（挑一两个有代入感的细节写实，其余略） |

**一动一静**：动作冲击段后接安静感知段。情绪最高点（打脸 / 反转）用「动」，最低点（心死 / 余韵）用「静」。不连续 2 节全动（暴力疲劳），不连续 2 节全静（拖沓）。

**主语节奏**：段首 / 场景切换 / 多人同场用主角名建立主语；同一动作链内用代词 / 省略流动；关键转折再点名。避免连续多句都以同一主角名开头。

## 12. 开头事件密度

前 100 字 ≥ 3 个事件。删掉所有背景介绍（「她是 XX 的女孩」「在古代 XX 年间」），第 1 句就是事件 / 对话 / 信息炸弹，不是描述。

**开篇第一段必须砸下一个可量化、能截图的「命级 / 羞辱级」赌注**——救命钱、被打掉的孩子、当众社死的实锤——三秒内把命 / 钱 / 孩子砸到脸上，不要慢热内心铺垫、不要单点反复抒情。赌注与细节要有**不可替换的辨识度**（具体到独特的数字 / 物件），别写谁都能套、换个名字就通用的桥段。

- 高密度（《闻笙有两意》原文开头）：老公怕我打搅他跟小秘书看演唱会，将我锁在家里。突发火灾，我打电话给他求救，只换来他一句：「能不能大度点，不就是拿你两张演唱会门票吗？」——三句砸下锁人、火灾、冷漠拒绝三件事。
- 低密度（要避免）：把开头花在「她是谁、家世如何、那天天气怎样」上慢慢铺。
