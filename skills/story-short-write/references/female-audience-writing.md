## 女频核心原则

- **安全感优先**：女频读者需要明确知道主角不会一直被虐，文案/简介中必须暗示成功或希望
- **主体性敏感**：女主如果缺乏主体性（被动、被安排、无选择权），吸引力和留存风险升高
- **情绪产品导向**：不论什么风格（沙雕/正剧/甜文），本质都是情绪产品，核心是精准满足目标读者的情感需求

## 文案结构体系

文案由**状态、困境/意外、行动、成功/失败**组合而成，至少包含其中两个。

| 类型 | 结构 | 适用场景 |
|------|------|----------|
| 二段式悬疑型 | 状态 → 困境/意外 | 悬疑文、留悬念 |
| 三段式行动型 | 状态 → 困境/意外 → 行动 | 展示金手指/计划 |
| 三段式成功型 | 状态 → 困境/意外 → 成功 | 甜文/轻松向/低复杂度文案常用 |
| 四段式成功型 | 状态 → 困境/意外 → 行动 → 成功 | 最主流结构 |

叠加技巧：
- 同一文案内叠加多次"成功"以增强爽感：`状态→困境→行动→成功→行动→成功`
- 不同梗叠加不同类型：`四段式成功 + 二段式悬疑`
- 开头用二段式悬疑 + 后面跟四段式成功，兼顾悬念和爽感

文案安全法则（需用当前扫榜/拆文样本验证）：
- 缺少女主成功剧情 → 读者缺乏安全感 → 吸引力下降
- 悬疑型文案叠加过多 → 读者理解不了 → 吸引力下降
- 文案和文名货不对板 → 劝退
- 女主被虐过重 → 整体偏虐 → 吸引力远低于预期
- 事业线突出的文 → 文案必须体现事业成功暗示

## 读者群体差异

| 读者类型 | 核心需求 | 偏好题材 |
|----------|----------|----------|
| 家庭主妇型 | "有资格抱怨"，男性有钱提供物质保障 | 强取豪夺、先婚后爱、追妻火葬场 |
| 事业型女性 | 男人是感情安慰，更需要情绪价值 | 独立女主、事业线+爱情线并重 |
| 学生群体 | 相信爱情战胜一切 | 纯爱、校园、靠爱情战胜艰难险阻 |

## 热门题材与梗

### 热梗使用原则

不保留历史风口题材清单。需要判断题材热度时，先用 `/story-short-scan` 或最新扫榜/拆文报告验证目标平台样本，再只抽取可复用功能位（身份优势、公共信息差、亲情保护、事业成功暗示、群体反应等），不要照搬旧题材壳。
### 感情线为主的类型
- **破镜重圆**：男女主相识于微末，分开后某方发迹，再遇爱恨拉扯
- **恨海情天**：双方/一方以为只有恨没有爱，慢慢揭露隐藏的爱意
- **酸涩感情流**：爱而不得的暗恋 / 感情沦陷内心挣扎
- **冲突后置文**：替嫁/和离/出宫/换亲/流放/抛夫弃子/断亲/死遁后的第XX年

## 扫榜与趋势验证方法

### 验证步骤
- 使用 `/story-short-scan` 或最新扫榜/拆文报告获取目标平台样本。
- 记录样本来源、日期、平台、题材、文案结构和读者反馈关键词。
- 排除自带粉丝基础或非目标平台的样本。
- 只提取可复用的结构功能：梗组合、文案安全感、情绪钩子、反转位置。

### 影响吸引力的关键因素
1. 梗的新鲜度：同平台近期样本越少，越需要用 scan 复核。
2. 文案质量：结构完整+爽点明确。
3. 题材热度：必须以当前 scan/analyze 样本为准。
4. 书名吸量：噱头十足且与正文承诺一致。

## 文案与人设避坑

### 人设塑造要点
- 女主人设用对比法效果显著：和配角（如恋爱脑角色）做对比，坚持自己路线来丰满人设
- 精准人设要有具体行为展现，如"烫头逃学装大款，抽烟喝酒打群架"比"叛逆少女"更有画面感
- 男主人设用细节描写加分：如"抛剑入鞘，血腥气跟着剑鸣一同在古殿中震颤荡开"
- 反差是人设灵魂：外表和内在的反差（清纯如天使却胆大妄为）、人设和用词的反差（古板人设但用热情词汇）
- 女主多心理描写用词活泼，男主多行动外表描写——分工明确人设更精准
- 只有总写没有具体描写→面目模糊；有整体描述但缺乏行动和心理→配角感
- 人设让读者有安全感很重要：暴发户女儿重生夺回自己的东西、美貌学霸重生过得好

### 文案常见坑
- 文名和文案内容必须一致，货不对板会劝退读者
- 文案中爽点和安全感必须明确呈现，女主的行动和成功不能缺失
- 虐渣复仇线不明朗会降低吸引力
- 金手指不明确会削弱安全感
- 文案过于降智出戏的设定会影响整体可信度
- 感情线发展缺乏张力时，即使梗好也难以留住读者
- 简介信息量足但条理要清晰，过于杂乱会让读者看着累
- 文名突出重点元素或噱头，与文案内容匹配

## 感情线写法深化

### 追妻火葬场变体
- 标准版：婚后过得不好→女主离开→男主后悔追回
- 带球跑版：女主带球跑→宝宝找爸爸→撮合父母
- 觉醒版：女主重生后退婚→前夫世后悔
- 反转版：女主以为是追男→实际是男追女的暗恋局

**虐情递进（付费点前）**：四阶段，一阶比一阶狠，为"女主心死、男主后悔"铺垫。

| 阶段 | 做法 |
|------|------|
| 日常漠视 | 用细碎日常放大女主的隐形付出与男主的理所当然——她准备的被随手搁置，她的难处他从不正眼看 |
| 当众冒犯 | 升级为公开羞辱、凡事偏向第三方，让女主的爱显得一文不值 |
| 毁其所重 | 亲手毁掉女主最珍视的人或梦想，断她回头的念想 |
| 平静离开 | 不哭不闹、异常平静地走——哀莫大于心死，平静比哭闹更有力量 |

三个加分技法：短回忆做甜虐对照（回忆越甜当下越虐）、女主态度从卑微到冷硬的强反差、第三方补刀。付费点后接「拉扯节奏」，收尾用「回旋镖」（见 `genre-writing-techniques.md`）把前期的伤一条条还回去。男主要"有缺陷但有被爱的资本"，不能只负面降智。追妻题材公式见 `genre-writing-formulas.md`，此处只给打法、不重述。

### 强取豪夺变体
- 身份差版：继父长子看上女主/侯爷看中婢女/外室上位
- 抢婚版：男主强取豪夺直接抢婚
- 伪亲属版：伪弟弟/舅舅的禁忌暗恋
- 反向版：男主为爱甘当替身、男主当外室

### 先婚后爱变体
- 换亲版：重生女来换亲→女主嫁冷面大佬→感情渐好
- 保姆版：穿成大佬保姆→日久生情→结婚
- 年代版：年代文嫁给军人/工人→婚后甜蜜
- 联姻版：和青梅竹马联姻→发现对方早有觊觎

### 暗恋/觊觎模式
- 阴湿暗恋：表面不和实际觊觎多年
- 伪亲属暗恋：假装姐弟/叔侄实则暗恋
- 双向暗恋：双方都以为对方不喜欢自己
- 日记暴露：女主日记被男主看到→你来我往

### 拉扯节奏：你追我逃 → 你悔我已走

虐恋/追妻的拉扯，根子是男女主的感情不在同一个时间点上：一方还在进，另一方已经退。三段递进：

| 阶段 | 女主 | 男主 | 读者情绪 |
|------|------|------|----------|
| 一·女主先抽离 | 默默收心、准备走 | 没察觉，仍傲慢"她离不开我" | 急着想看男主翻车 |
| 二·男主紧追 | 一退再退、平静陌生 | 后知后觉、卑微试探、全城找人 | 解气，盼女主硬气 |
| 三·男主求而不得 | 事业新生、彻底翻篇 | 困在悔恨、换不回一个回眸 | 爽感顶点 |

- **名场面**：挑 1-2 个反差场景做足——雨夜守门被保安赶走、当众一句"这位先生，我们认识吗"、下跪哀求换来"当初你伤我时，怎么没想过今天"。
- **对照角色助攻**：放一个更优秀、更尊重女主的男二自然站在她身边，坐实男主"局外人"身份；男主的嫉妒疯魔对上女主的从容，拉扯感拉满。

### 感情线爽点要素
- 反差萌制造火花：斯文古板×胆大妄为、高岭之花×肆意张狂
- 虐的目的是制造情绪波动，不是为虐而虐——虐完必须有反转或甜
- "失败型"文案在感情线中可以是成功：女主视角的失败=感情线更进一步
- 前期甜→中期虐→后期火葬场追回，是验证过的经典节奏
- 双视角（男女主各自视角）增强感情张力

## 热门题材细分类

### 穿书/穿原创变体
- 穿成炮灰女配/反派/路人→觉醒改命
- 穿成男主的保姆/母亲/姐姐→身份反转
- 穿成苦情文女主母亲→发疯改命
- 穿到游戏/克系/星际世界→新奇设定
- 穿书+系统金手指+吃瓜群众→轻松向改命

### 年代文细分
- 七零八零九零各有受众，八零最热
- 换亲+年代文+先婚后爱是经典组合
- 重生回到高考前→改变命运；热度需用当前 scan/analyze 验证
- 穿成年代文暴发户炮灰→觉醒阻止悲剧
- 和闺蜜一起穿成妯娌→拼命捞钱→搞笑
- 年代文核心：极品家人打脸+事业奋斗+甜蜜感情

### 快穿/系统文细分
- 每个世界独立，适合卖不同类型的CP和梗
- 快穿+表小姐+心机上位→古言穿多世界
- 系统绑定的可以是物品（写文系统）而非人
- 快穿世界古今混合会劝退部分读者，最好同一类型

### 事业线类型
- 经商种田：种田+卖货+发家致富
- 娱乐圈：退圈转行/影后重生/参加选秀
- 体育竞技：乒乓球/体操等带队走出低谷
- 科研学术：科研大佬的漂亮原配
- 美食文：做菜好吃→吸引各路人→事业成功
- 写文/自媒体：在异世界写文封神
- 萌宠相关：养熊猫/养动物→意外爆红

## 女频vs男频关键差异

| 维度 | 女频 | 男频 |
|------|------|------|
| 核心驱动 | 情绪产品，几乎都含感情线 | 剧情爽感，纯感情线罕见 |
| "剧情流爽文"定义 | 与男频不同，废土/赛博是小众 | 升级打怪是主流 |
| 读者对异性角色重视度 | 极高，男主人设决定留存 | 较低，女主功能性更强 |
| 虏的目的 | 制造情绪波动，虐完必须有反转 | 升级打怪的前置铺垫 |
| 复仇方式 | 信息差式（真相揭露打脸） | 力量式（实力/身份碾压） |
| 情绪触发密度 | 3.23次/千字 | 2.50次/千字 |
| 反转密度 | 1.77次/千字 | 1.21次/千字 |
| 反转类型 | 信息差型（心声/真相） | 力量型（身份/实力） |

跨频规则：女频读者更在意角色间的情感张力和互动细节，女频事业线需要搭配感情线才更容易出成绩。

## 趋势候选功能位（需实时验证）

使用前先用 `/story-short-scan` 或最新扫榜/拆文报告确认目标平台仍有样本支撑；确认后只抽功能位，不照搬题材壳。

| 候选方向 | 可复用功能位 |
|---|---|
| 反霸总/反恋爱脑 | 传统权力幻想反转 + 女主主体性 |
| 身份反转 | 低位身份误判 + 高位真相揭示 |
| 双女主/闺蜜穿书 | 双主角协作 + 互补能力 + 共同目标 |
| 年代文框架+现代价值观 | 历史处境压力 + 现代认知差 |
| 极品家人打脸+自我觉醒 | 不公平伤害 + 主体性觉醒 + 报应兑现 |

## 质量检查

写女频故事时对照：

- [ ] 文案含女主成功暗示（不缺安全感）？
- [ ] 女主有主体性（主动选择，不被安排）？
- [ ] 男主人设用细节描写加分（不只形容词）？
- [ ] 感情线有张力（修罗场/反差萌/虐后反转）？
- [ ] 文名和文案内容一致（货不对板劝退）？
- [ ] 金手指明确（安全感基础）？
- [ ] 题材组合的热度已用最新 scan/analyze 或用户指定平台信息验证？
- [ ] 目标平台写法与文风定位匹配？
