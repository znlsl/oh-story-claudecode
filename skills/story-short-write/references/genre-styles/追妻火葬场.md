# 题材风格包：追妻火葬场 / 虐恋意难平

> 番茄 / 七猫 / 黑岩通俗短篇。先读 `short-craft.md` 通用底座，本包在其上覆盖该题材专属调性。
> 一句话定位：**主角被最亲密的人（丈夫 / 未婚夫 / 父母）为「白月光」牺牲，一步步折磨到心死，冷静决裂离场；他后悔莫及、追悔火葬场。** 读者要的是「被珍视感的彻底剥夺」+「心死后的扬眉吐气」。

---

## 叙述腔调

第一人称「我」受害者全程主导，贴体感和心跳走，毫不克制——疼就说疼，心死就说心死。双轨：前半委屈宣泄、眼泪决堤；撞破真相后切成冷静、反讽、居高临下的审判腔。句子短、白、口语，大量直给的内心独白和反问。**允许并鼓励向前剧透「火葬场」结局来吊人。**

开口就是带占有关系的称呼：「我老公 / 我丈夫 / 我未婚夫」。

---

## 开篇范式

正文第 1 章前先放一段**结果倒序的黄金简介**（150-300 字），把背叛全貌 + 心死 + 「可后来他后悔了 / 他疯了」一次性剧透，专卖追读；正文第 1 章把简介事件慢放重写、补满对话体感。

| 开篇式 | 真实范例 |
|--------|----------|
| 关系+年限锚点 | 《敦宜》和裴筠舟成亲的第七年，他带回了一个罪奴，是他的白月光。 |
| 第一句即背叛炸弹 | 《此后明月隔山海》第一次发现丈夫沈浩和别人滚在床上，是在我们的婚房里。 |
| 身份+反差自陈 | 《五年恩情，一朝两清》我是许念安，是顾泽言结婚证上的另一半，也是他口中用恩情绑架婚姻的女人。 |
| 灾难求救被辱 | 《闻笙有两意》老公怕我打搅他跟小秘书看演唱会，将我锁在家里。（火灾求救换来「能不能大度点」） |
| 反差身份钩 | 《深情何时落暖阳》我给傅深当了八年替身。 |

## 钩子母题与密度

密度极高：简介塞 3-5 个背叛事件，正文每节至少一个新背叛升级或一句更狠的台词，章尾必留新炸点。**阶梯式背叛：道德 → 经济 → 生命。**

- **白月光触发链**（核心母题，几乎每篇都有）：男人有个白月光 / 小青梅 / 前妻 / 义妹，关键时刻为她牺牲妻子。白月光多用「娇弱委屈」人设当软刀。
- **偷听 / 撞破真相一次性砸**：无意听见渣男和同伙 / 白月光的真心话，全盘真相一次性砸下，是「心死」的导火索。
- **灾难期求救反被辱**：火灾 / 地震 / 绑架 / 早产时向丈夫求救，换来冷漠或咒骂，一步坐实「他根本不爱你」。
- **朋友圈配文当刀**：渣男 / 白月光发的甜蜜配文被主角刷到，配文越深情，刀越钝越疼。
- **数字 / 年限承重再清零**：结婚七年、陪你十年、八年替身——堆出投入再被一句话清零。
- **火葬场前瞻预告**：简介结尾或男视角插一句向前剧透 payoff（「我还不知道，他正发疯似的在国内找我」）。

## 情绪烈度与模式（烈度 9）

前半直白宣泄 + 撞破真相后冷静审判（钝刀转冷刀）。

- 直接命名情绪并加重：心如死灰、心彻底凉透、伤心欲绝、泪如雨下、如遭雷劈——通篇高频直给。
- 直白情绪 + 身体反应**叠加**（不是二选一）：「指甲深深扣进掌心」同段配「心如刀绞，几欲作呕」。
- 内脏 / 心脏拟痛：「巨大的悲痛在一点点啃食我的心脏，痛到不能呼吸」。
- 心死后转冷反衬：「这些话出自最亲密的人之口，却已经激不起我任何波澜」。

真实范例：

- 《闻笙有两意》我的眼泪不争气地往下掉，不知道是委屈还是疼的。
- 《我替丈夫白月光顶罪后》即使早已心死，心脏依然狠狠抽痛。原来在他心里，我连只猫都不如。
- 《为给白月光出气，男友群发我私密照》随着我一丝不挂的照片被他扔在茶几上，我的心也跟着沉到了谷底。

## 对话风格与真实金句

占比 45-60%，口语、狠，独立成段。三类台词撑全篇：

**① 施害者道德踩踏狠话（钩子本体）**
- 能不能大度点，不就是拿你两张演唱会门票吗？非要跟人家小姑娘过不去。
- 婉婉身子弱，受不了这种惊吓。十八楼高而已，反正下面有气垫，老婆，你不会怪我的，对吗？
- 演得像一点，别丢顾家的脸。

**② 白月光娇弱软刀**
- 对不起，笙笙姐姐，都怪我非要让池哥哥陪我去看演唱会。
- 我把小汀当妹妹，她喜欢你的眉眼，你割点眉毛和眼睫毛又不会死？

**③ 主角心死后冷静审判金句**
- 结婚七年，你连我穿多大码的鞋都不知道，再在一起，就再没必要了。
- 如你所愿，生不如死了。
- 秦总误会了，我纯粹觉得你比较可笑。

冲突靠对话长度反转标记权力易主：渣男前期长篇 PUA，主角后期一句话冷怼。

## 节奏骨架

```
黄金简介（结果倒序 + 火葬场预告）
→ 第1章重写简介事件、补满对话体感
→ 逐章背叛升级（道德→经济→生命）
→ 偷听 / 撞破真相
→ 心死定格
→ 冷静审判 + 净身出户 / 反击
→ 渣男追悔 / 发疯（火葬场）
```

每章一个新信息炸弹，章尾留钩不收束情绪。疏密：背叛 / 打脸 / 反转处详写慢放，过场一句带过。

## 收尾范式

- 火葬场反转：主角走后渣男后悔莫及 / 发疯 / 求原谅，主角已无波澜或冷拒。
- 转身求婚 / 改嫁他人，当众完成关系翻转。
- 带孩子 / 带资产一去不归，开启新人生。
- 反讽对位：用渣男视角写他失去后的崩溃，漠视比报复更狠。

真实范例：《敦宜》破镜本难圆……我绕开他们，头也不回地往自家骆驼走去。／ 多篇简介定式收口：可后来，他却后悔了。

## 该题材专属调校（与通用去 AI 规则的偏差）

这些在别的语境是 AI 味，在本题材是卖点，**别误删**：

- **直写情绪成语**（心如死灰 / 泪如雨下）：保留，配一个场景里特有的动作或物件即可。别替换成纯动作。
- **第一人称成段宣泄独白 + 连环反问**：保留，这是代入与共鸣的主引擎。
- **火葬场前瞻预告 / 结果倒序简介**：保留这类有意识的向前剧透；只删「泄了悬念又不吊读」的随手上帝腔。
- **审判金句 / 心死式章尾**：保留；只删空洞哲理升华。

仍要避免的真·AI 味：映入眼帘、心中暗道、沉声道、嘴角微扬、目光如炬、论文体连词（不难看出 / 由此可见）、空泛无体感的情绪总结句（一丝悲伤涌上心头）。

> 标点：本题材范例为示意保留了原貌；落到正文按 `short-craft.md` 第 8 节执行（不出 `……` / `——`，停顿用短句 / 动作 / 换行）。

## 配套结构公式（按篇幅选）

- **现代追妻 / 不原谅型（8-11 章）**：1-2 章虐（伤害言语→行为→身体升级）→ 3-4 章心死离开（临界事件）→ 5-6 章真相 / 反转 → 7-8 章渣男追悔被拒 → 治愈 / 新生活暗示。100% 第一人称；伤害对话「精准刺痛」非谩骂；觉醒是一瞬间；爽按序释放。
- **重生 / 死遁型**：先交代前世死法与凶手 → 再睁眼回到关键节点 → 做「前世绝不会做的事」（主动把丈夫 / 房子送给白月光，或决绝死遁）→ 渣男追悔。
