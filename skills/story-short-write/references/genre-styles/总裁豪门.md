# 题材风格包：总裁豪门 / 现代言情虐

> 番茄 / 七猫 / 黑岩通俗短篇。先读 `short-craft.md` 通用底座，本包在其上覆盖该题材专属调性。
> 一句话定位：**豪门妻 / 联姻太太被丈夫为「白月光 / 小三」一步步背叛，物证慢推坐实出轨，一步步折磨到心死；主角从发疯闹到冷处理离场，靠弹幕 / 系统 / 重生外挂拿到「不必再舔」的清醒理由，最后净身出户 + 经济反杀，渣男追悔火葬场。** 读者要的是「被珍视感剥夺」+「道德绑架被一句话戳穿」+「清醒后扬眉吐气」。

与「追妻火葬场」同根（白月光 / 偷听 / 心死 / 火葬场都通用），本包加味处在：**渣男道德绑架金句、物证慢推坐实、发疯→冷处理的心死开关、外挂给清醒、豪门经济 / 身份反制**。这五样是本题材的招牌，别只复刻追妻。

---

## 叙述腔调

第一人称「我」豪门妻 / 未婚妻全程主导，贴体感和心跳走，疼就说疼。开口带占有称呼：「我老公 / 我丈夫 / 我未婚夫 / 顾太太」，并嵌豪门设定锚（集团、股东大会、联姻、恩情、京圈、继承权）。

双轨切换，但本题材的切换更「狠」：前段不只委屈宣泄，常是**真发疯**（当众开撕、网上对线、砸东西）；撞破真相后切成**冷处理**——不闹、应一声「好」、看戏、记账。冷处理段大量反讽和吐槽。

带外挂时多一条**吐槽声线**：弹幕 / 系统 / 脑海声音 / 重生记忆插话，给主角清醒判断，也给读者爽点。这条声线口语、网感、可以贱，别雅化。

允许并鼓励向前剧透「火葬场」结局来吊人。

## 开篇范式

正文第 1 章前先放一段**结果倒序的黄金简介**（150-300 字），把背叛全貌 + 心死 + 「可后来他后悔了 / 他疯了」一次性剧透；正文第 1 章把简介事件慢放重写、补满对话体感。本题材简介常多塞一层「外挂 / 身份底牌」当钩。

| 开篇式 | 真实范例 |
|--------|----------|
| 灾难求救被道德绑架 | 《闻笙有两意》老公怕我打搅他跟小秘书看演唱会，将我锁在家里。突发火灾，我打电话给他求救，只换来他一句，「能不能大度点，不就是拿你两张演唱会门票吗？」 |
| 身份自陈 + 恩情绑架 | 《五年恩情，一朝两清》我是许念安，是顾泽言结婚证上的另一半，也是他口中用恩情绑架婚姻的女人。 |
| 冷战分手后渣男普信通知 | 《心死准备离场，冷战男友通知我领证》分手三年后，突然收到秦衍的消息：「许昭昭，三年了也该消气了，今晚我回国你来接我，带上你爸妈，我们商量一下领证时间。」 |
| 物证开篇（明知过敏还送花） | 《情人节的第二束花》情人节，收到丈夫送来的两束花，一束干花，一束鲜花，可他明知道我鼻炎，闻不得花香。 |
| 重生 / 外挂自陈 | 《弹幕教我做恶毒千金后，竹马离不开我了》重来一世，我头上突然多了几行弹幕。【周迟欢，你要时刻记住！舔狗舔狗，舔到最后一无所有！】 |
| 豪门股权局开场 | 《双胞胎梗？我都玩烂了》我病危后，集团立刻召开股东大会打算易主给女儿。丈夫却带私生女出现，大言不惭的说当年其实我生的是双胞胎。 |

## 钩子母题与密度

密度极高：简介塞 3-5 个背叛事件，正文每节至少一个新背叛升级或一句更狠的台词，章尾必留新炸点。**阶梯式背叛：道德 → 经济 → 生命**，本题材尤其爱把「经济」做足（集团、股权、继承权、净身出户）。

- **道德绑架链**（核心母题）：渣男把背叛包装成「你该大度 / 她可怜 / 你欠她 / 你该感激我」，越理直气壮越是钩。
- **白月光 / 小三触发链**：丈夫有白月光 / 小青梅 / 战友之女 / 学妹，关键时刻为她牺牲妻子；白月光用「娇弱委屈」当软刀，常配当众晕倒 / 见血反咬。
- **物证慢推坐实出轨**（本题材招牌，区别于一次性偷听）：不一次砸，靠一件件物证累积——干花 / 杯中倒影 / 香水味 / 一根头发 / 不该有的试用装——读者陪主角一步步验证，钝刀更疼。
- **朋友圈 / 大屏配文当刀**：甜蜜配文 + 信物错位（主角送丈夫的表戴在白月光腕上 / 两人上了演唱会大屏），配文越深情，刀越钝越疼。
- **灾难期求救反被辱**：火灾 / 脑溢血 / 丧父时求救，换冷漠或「我在开会」「家属请回避」。
- **数字 / 年限 / 恩情承重再清零**：结婚五年、在一起八年、三年协议、用恩情绑架——堆出投入再一句话清零。
- **外挂给清醒理由**：弹幕 / 系统 / 重生 / 脑海声音，给主角「这次不舔」的依据，兼供吐槽。
- **火葬场前瞻预告**：简介结尾或男视角插一句剧透 payoff（「他却在某个深夜，疯了一样砸开我的公寓」）。

## 情绪烈度与模式（烈度 9，双段：发疯 → 冷处理）

本题材心死开关有明确两段，烈度全程不降，只换形态。

- **前段（发疯 / 宣泄）**：直白命名情绪并加重，眼泪不争气、心如死灰、浑身发凉；可以当众开撕、网上对线、砸东西。直写情绪 + 身体反应**叠加**（指甲掐掌 / 几欲作呕 / 脊背发凉）。
- **后段（冷处理 / 审判）**：撞破真相后切冷，不闹、应一声好、看戏。情绪不消，转成「冷烈」——反讽、吐槽、记账。心死后转冷反衬最杀：「这些话已经激不起我任何波澜」。
- 外挂带来的清醒，让后段冷得有理有据，不显突兀。

真实范例：

- 《闻笙有两意》我的眼泪不争气地往下掉，不知道是委屈还是疼的。
- 《白许空诺爱成沫》我脊背发凉，指甲掐着掌心抑制住浑身颤抖，拨通了那个没有备注的电话。
- 《五年恩情，一朝两清》从那天起，我好像心不再那么疼了。但心里某个地方，彻底安静了下去。
- 《男朋友是PUA高手》我从一开始的发疯到后来的视若无睹，甚至还会开车送他和别的女人去酒店。

## 对话风格与真实金句

占比 45-60%，口语、狠、独立成段。四类台词撑全篇（本题材比追妻多一类「外挂吐槽」）：

**① 施害者道德绑架金句（钩子本体，越理直气壮越对）**
- 能不能大度点，不就是拿你两张演唱会门票吗？非要跟人家小姑娘过不去，还编出这么拙劣的谎言。（《闻笙有两意》）
- 你在网上乱发什么，知不知道这叫网暴，浅浅她有抑郁症，现在所有人都在骂她是小三。（《闻笙有两意》）
- 演得像一点，别丢顾家的脸。（《五年恩情，一朝两清》）
- 许念安，我现在没空处理你的情绪。有事找助理。（《五年恩情，一朝两清》）
- 你大度点嘛，男人三妻四妾很正常。（《心死准备离场，冷战男友通知我领证》）
- 昭昭，如今许家败落，海城圈子里谁会愿意娶你，你应该感激我！（《心死准备离场》）
- 我有私生女还不是因为你这个工作狂？你根本不关心家庭，没有尽到做妻子的义务！所以我才会出去寻找感情寄托！（《双胞胎梗？我都玩烂了》）
- 作为医生，我要避嫌，家属请回避。（《医生老公为了避嫌不做手术，可死的是他妈啊》）

**② 白月光 / 小三娇弱软刀**
- 对不起，笙笙姐姐，都怪我非要让池哥哥陪我去看演唱会，还上了大屏。（《闻笙有两意》）
- 爸爸，这样不好，还是让姐姐当总裁吧……（《双胞胎梗？我都玩烂了》）
- 你别怪阿衍哥哥，他们男生就是这样心大。阿衍哥哥爱你，我只要看着他幸福就够了！（《心死准备离场》）
- （当众委屈，私下挑衅）贱人，被那群男人玩得开心吗？（《为讨真千金欢心，我被送去夜总会》）

**③ 主角心死后冷静审判 / 反讽金句**
- 秦总误会了，我纯粹觉得你比较可笑。（《心死准备离场》）
- 这世界上没有谁非谁不可！更何况三年过去了，我早已结婚生子。（《心死准备离场》）
- 你确定你喝的是六个核桃不是六个核弹？（《心死准备离场》）
- 我只有一个女儿。（《双胞胎梗？我都玩烂了》）
- 你是猪脑袋还是当我傻？每次产检的B超都只有一个孩子。（《双胞胎梗？我都玩烂了》）
- 随便玩，但别弄出私生子。（《老公被骗光钱后，我拿走了他的一切》）

**④ 外挂吐槽声线（弹幕 / 系统 / 心声，网感、可以贱）**
- 【舔狗舔狗，舔到最后一无所有！】【把他骂爽了，命都给你！】（《弹幕教我做恶毒千金后，竹马离不开我了》）
- 这男主也太普信了吧，长嘴不如不长。（《心死准备离场》弹幕）
- 渣男配贱女，别来纠缠我！（《心死准备离场》心声）
- 宿主，自从你走后，剧本内没了重要女配世界都乱套了！趁主角团还没察觉，你先回去顶一阵。（《风止意已平》系统）

冲突靠对话长度反转标记权力易主：渣男前期长篇道德绑架 / 普信车轱辘话，主角后期一句话冷怼。

## 收尾范式

- 火葬场反转：主角走后渣男后悔莫及 / 砸门 / 发疯找人，主角已无波澜或冷拒。
- 净身出户 / 经济反杀：签好协议递过去、掏空集团成全「负翁」、亮股权或 CEO 身份碾压。
- 转身改嫁 / 已婚生子：渣男普信归来才发现主角早结婚生子。
- 漠视收尾：应一声好、不再纠缠，比报复更狠。

真实范例：

- 《五年恩情，一朝两清》直到我签好离婚协议，安静地消失在他的世界。
- 《双胞胎梗？我都玩烂了》想要集团就拿去吧，反正这些年我也把资产搂空了。
- 《浮生暂寄梦中梦》可后来，我却听人说，疯了。衣衫不整的跪在大街上，呼喊着我的名字。
- 《一江烟水照晴岚》可后来，他红了眼眶，怎么找，也找不到回家的那段路。

## 该题材专属调校（与通用去 AI 规则的偏差）

这些在别的语境像 AI 味或可删冗余，在本题材是卖点，**别误删**：

- **道德绑架金句**：理直气壮、逻辑越离谱越对（「她有抑郁症你要大度」「我出轨是你工作狂逼的」）。别改顺、别加旁白替读者点破——读者要的就是看渣男自己说出口。
- **物证侦探式内心推理段**：放大照片、查品牌有没有赠送活动这类「自证不是多想」的独白，是慢推引擎，别当啰嗦删；删了刀就钝了。
- **弹幕 / 系统 / 心声吐槽**：网感、口语、可以贱（「长嘴不如不长」「六个核弹」），别雅化、别统一成书面语。
- **普信渣男的车轱辘话与重复**（「我只爱你，宁宁是爱我我不娶她」反复说）：是刻意的，体现 PUA 钝刀和主角的无力感，别当重复冗余砍。
- **直写情绪成语**（心如死灰 / 浑身冰冷）：保留，配一个场景里特有的动作或物件即可，别替换成纯动作。
- **火葬场前瞻预告 / 结果倒序简介**：保留这类有意识的向前剧透；只删「泄了悬念又不吊读」的随手上帝腔。
- **审判金句 / 心死式章尾**：保留；只删空洞哲理升华。

仍要避免的真·AI 味：映入眼帘、心中暗道、沉声道、嘴角微扬、目光如炬、论文体连词（不难看出 / 由此可见）、空泛无体感的情绪总结句（一丝悲伤涌上心头）。
