# 知乎盐言故事风格参考

> 写/拆知乎盐言故事时加载。先确认盐言风格特征，再用检查项逐条对照。

## 决策路由

| 你的场景是 | 用这个部分 | 操作 |
|------------|-----------|------|
| 写盐言故事 | 盐言核心风格特征 + 类型情绪模式 | 严格限定第一人称、`「」` 对话 |
| 拆盐言故事 | 拆文时盐言检查项 | 逐条检查视角/格式/句式/反转位置 |
| 对比平台差异 | 平台风格差异 | 知乎 vs 番茄 vs 小红书三列对比 |

## 盐言核心风格特征

| 特征 | 规则 |
|------|------|
| 视角 | 第一人称限定，不出现全知描述 |
| 对话格式 | 使用 `「」`，独立成段，说话人标签简短 |
| 句式 | 长短句结合，叙述以逗号长句为主，短句只落在情绪节点；句子内部不碎，忌通篇短句的电报体 |
| 排版 | 段首两个全角空格，段落间无空行，章节间一个空行 |

## 拆文时盐言检查项

| 检查项 | 说明 |
|--------|------|
| 视角一致性 | 全程第一人称，有无视角跳跃 |
| 对话格式 | 统一 `「」`，无混用引号 |
| 句式节奏 | 长短是否交错：叙述以逗号长句为主、短句只落情绪节点，无通篇短句的电报体 |
| 口语程度 | 有无书面腔/翻译腔残留 |
| 情绪节奏 | 有无明确情绪曲线（盐言最看重情绪拉扯） |
| 反转位置 | 是否在 70-85% 位置（盐言标准位置） |
| 章尾钩子 | 每章结尾是否有钩子 |

## 盐言类型与情绪模式

| 类型 | 核心情绪 | 典型反转 |
|------|----------|----------|
| 虐恋 | 意难平 | 他其实一直爱她 |
| 重生 | 爽感释放 | 这一世不再犯同样的错 |
| 悬疑 | 细思极恐 | 凶手就在身边 |
| 职场 | 共鸣愤怒 | 反杀成功 |
| 婚姻 | 共鸣+治愈 | 离开之后更好 |
| 治愈 | 温暖 | 被忽视的爱被发现 |

## 平台风格差异

| 维度 | 知乎盐言 | 番茄短篇 | 小红书故事 |
|------|----------|----------|------------|
| 视角 | 第一人称限定 | 不限 | 不限 |
| 篇幅 | 8000-20000字 | 灵活 | 1000-5000字 |
| 核心卖点 | 情绪拉扯+反转 | 爽感节奏 | 话题性+共鸣 |
| 对话风格 | `「」` 盐言引号 | 引号 | 自由 |
| 读者期待 | 被虐到想哭反转治愈 | 一口气读完很爽 | 有话题想转发 |
| 付费模式 | 会员/单篇付费 | 免费阅读 | 免费引流 |
